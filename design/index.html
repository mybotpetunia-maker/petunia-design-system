System Contract

What Holograph is and what it is not.

What it is
A complete visual language for AI agents and background systems. Dark, precise, iridescent. Built with semantic tokens, a minimal but expressive component set, and Petunia's voice. WCAG AA throughout.
What it is not
Not a consumer-friendly design system. Not cheerful or approachable. Not a button library. Not opinionated about your product architecture. Not a redesign tool — it's a starting point for systems that need to be precise, quiet, and human.
Iridescent
Signature Accent

The iris gradient is the single most important visual element. One animated instance per screen. Reserve it.

Typography

Three font families, their roles, size scale, and the Bodoni dazzle zone rules.

Display Typeface
Bodoni Moda — weight 400, italic optional
The iridescent hero type
H1 hero at ≥48px, weight 400 only. Never below 48px at weight 400. The dazzle zone (18px–47px at weight 300–400) is forbidden.
H2 — Bodoni Bold
Bodoni Moda — weight 700 at 28–47px
Bold kills hairlines
At 28–47px, use weight 700 ONLY. Bold suppresses the hairline vibration that causes dazzle.
Body & Heading Typeface
IBM Plex Sans
All prose, all sub-headings (H3+), all UI copy. Legible at any size. Default body: 16px / 1.55 line height.
H3 and below always IBM Plex Sans at weight 500–600. Never Bodoni for H3.
Monospace Typeface
IBM Plex Mono
var(--color-teal-300) · 0.75rem · token-names · data-values · 2,841 · 14:32
Every number, every label, every data value. Uniform stroke weight. Never for running prose.

Bodoni Dazzle Zone (Strict)

Bodoni's extreme thick-thin strokes create visual vibration at mid-sizes. Your visual cortex processes competing spatial frequencies, causing dazzle.

  • ≥48px at weight 400 — hairlines become dramatic features
  • 28–47px at weight 700 ONLY — bold suppresses hairlines
  • Never weight 300–400 between 18px–47px
  • Never Bodoni for numerics — use IBM Plex Mono always

08

Agent Rules

Hard rules for building any Holograph surface. When uncertain, conservative wins.

Rule 01
Never hardcode color values
Always reference --color-* or semantic tokens. Zero exceptions.
Rule 02
Three fonts, strict roles
Bodoni 400 → H1 hero ≥48px only. Bodoni 700 → H2 at 28–47px. Plex Sans 500–600 → H3+. Plex Sans 400 → body. Plex Mono → ALL numerics.
Rule 03
Bodoni dazzle zones
Never Bodoni weight 300–400 between 18px–47px. Use 700 at 28–47px. Use 400 at ≥48px. Never Bodoni for numerics.
Rule 04
Text contrast minimum
Body prose min: --text-secondary (8.9:1 AAA). Labels ≥14px: --text-tertiary (5.4:1 AA) OK. Never --text-tertiary for running copy.
Rule 05
One iris element per view
The iridescent gradient is the hero accent. One animated iris element per screen. Reserve it for the single most important visual moment.
Rule 06
Teal is the primary action colour
Amber = caution. Violet = depth. Green = success. Red = error. Never amber for primary CTAs. When uncertain: teal.
Rule 07
Fade-up for all entrances
Every element entering uses fade-up: opacity 0→1 + translateY(16px)→0. Stagger lists 40ms. Always honour prefers-reduced-motion.
Rule 08
When spacing is uncertain, go larger
Generous negative space reads as precision. Min padding: space-5 (20px). Never space-1 or space-2 as sole padding.
Rule 09
Radius is consistently sharp
Default: radius-md (6px) for cards/inputs. radius-sm (3px) for badges. Never radius-xl for cards. Full radius only for avatars and toggles.
Rule 10
Always set background on body
Every Holograph surface must declare background-color: var(--bg-base) on root. Never allow white to show.
Rule 11
Copy is Petunia's voice
She does not say "Great!", "Certainly!", or "Let me know!" Cut copy in half, make it declarative, remove exclamation marks.

Do This, Not That

DO
  • Use Bodoni 400 only at H1 hero sizes (≥48px)
  • Use Bodoni 700 for H2 at 28–47px
  • Use IBM Plex Sans 500–600 for H3 and below
  • Use IBM Plex Mono for every numeric
  • Pair Bodoni H1 with mono eyebrow above
  • Wrap animations in prefers-reduced-motion
NEVER
  • Use Bodoni at any weight for numbers
  • Use Bodoni 300–400 between 18px and 47px
  • Use Bodoni for H3 or sub-headings
  • Use IBM Plex Mono for body copy
  • Use --text-tertiary for body prose
  • Hardcode any hex value
