01

System Contract

What Holograph is and what it is not.

What it is
A complete visual language for AI agents and background systems. Dark, precise, iridescent. Built with semantic tokens, a minimal but expressive component set, and Petunia's voice. WCAG AA throughout.
What it is not
Not a consumer-friendly design system. Not cheerful or approachable. Not a button library. Not opinionated about your product architecture. Not a redesign tool—it's a starting point for systems that need to be precise, quiet, and human.

Typography

Three font families, their roles, size scale, and the Bodoni dazzle zone rules.

Font Families

Display Typeface
Bodoni Moda
The iridescent
hero type

H1 hero at ≥48px weight 400 only. Never below 48px. Never for numbers. The dazzle zone (18px–26px) is forbidden for Bodoni.

Body & Heading Typeface
IBM Plex Sans
All prose, all sub-headings, all UI copy. H2 at 28–47px weight 700 (bold kills hairlines). H3+ always Plex Sans. Legible at any size.

Default body: 16px / 1.55 line height. Comfortable, not airy.

Monospace Typeface
IBM Plex Mono
var(--color-teal-300) · 0.75rem · token-names · data-values · status-codes

Every number, every label, every data value. Uniform stroke weight. Never for running prose.

Bodoni Dazzle Zone Rules (Strict)

Bodoni Moda creates visual vibration when its extreme thick-thin strokes compete at mid-sizes. Your visual cortex processes high and low spatial frequency signals simultaneously, causing dazzle.

  • ≥48px at weight 400 — hairlines become dramatic features
  • 28–47px at weight 700 ONLY — bold suppresses hairlines
  • Never weight 300–400 between 18px–26px for any reason
  • Never Bodoni for numerics — use IBM Plex Mono always

Capitalisation Rules

Mono eyebrow labels — ALL CAPS always. (e.g. RUN · 0047 · OUTREACH)

Button labels — Sentence case. Never Title Case. (e.g. "Save goal")

Navigation items — Sentence case. (e.g. "Run log")

Headings (H1–H3) — Sentence case. Never Title Case.

Toast / alert titles — Sentence case. Short. One clause maximum.

Form labels — Title Case for single words, Sentence case for multi-word. (e.g. "Goal", "Target date")

Em dashes — Use freely in Petunia's voice. They are her pause, her pivot.

Agent Rules

Hard rules for building any Holograph surface. When uncertain, conservative wins.

Rule 01
Never hardcode color values
Always reference --color-* or semantic tokens. Zero exceptions.
Rule 02
Three fonts, strict roles
Bodoni 400 → H1 hero ≥48px only. Bodoni 700 → H2 at 28–47px only. IBM Plex Sans 500–600 → H3+. IBM Plex Sans 400 → body prose. IBM Plex Mono → ALL numerics.
Rule 03
Bodoni dazzle zones
Never Bodoni weight 300–400 between 18px–26px. Use 700 at 28–47px. Use 400 at ≥48px. Never Bodoni for numerics.
Rule 04
Text contrast minimum
Body prose minimum: --text-secondary (8.9:1 AAA). Labels at ≥14px: --text-tertiary (5.4:1 AA) acceptable. Never --text-tertiary for running copy.
Rule 05
One iris element per view
The iridescent gradient is the hero accent. One animated iris element per screen. Reserve it for the single most important visual moment.
Rule 06
Teal is the primary action colour
Amber signals caution. Violet signals depth. Green signals success. Red signals error. Never use amber for primary CTAs. When uncertain: teal.
Rule 07
Fade-up for all entrances
Every element entering uses fade-up: opacity 0→1 + translateY(16px)→0. Stagger lists 40ms per item. Always honour prefers-reduced-motion.
Rule 08
When spacing is uncertain, go larger
This system favours generous negative space — it reads as precision, not emptiness. Minimum padding: space-5 (20px). Never space-1 or space-2 as sole padding.
Rule 09
Radius is consistently sharp
Default radius: radius-md (6px) for cards and inputs. radius-sm (3px) for badges. Never radius-xl for cards. Full radius only for avatars and toggle tracks.
Rule 10
Always set background on body
Every Holograph surface must declare background-color: var(--bg-base) on the root element. Never allow white to show through.
Rule 11
Copy is Petunia's voice, not a chatbot's
Every word is Petunia speaking. She does not say "Great!", "Certainly!", or "Let me know if you need anything else!" Cut copy in half, make it declarative, remove exclamation marks.

Quick Reference — Do This, Not That

✓ DO
  • Use Bodoni 400 only at H1 hero sizes (≥48px)
  • Use Bodoni 700 for H2 headings at 28–47px
  • Use IBM Plex Sans 500–600 for H3 and below
  • Use IBM Plex Mono for every numeric value
  • Pair a Bodoni H1 with a mono eyebrow label above it
  • Wrap all animations in prefers-reduced-motion
✗ NEVER
  • Use Bodoni at any weight for numbers or data
  • Use Bodoni weight 300–400 between 18px and 47px
  • Use Bodoni for H3 or sub-headings
  • Use IBM Plex Mono for running body copy
  • Use --text-tertiary for body prose
  • Hardcode any hex value — always use tokens
